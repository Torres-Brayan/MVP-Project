WELCOME TO MY FAVORITE SOCCER PLAYERS API!

To access all the favorite soccer player list use the /favorite/players route.

To access a single favorite soccer player list use the /favorite/players/:id route.

To create a favorite soccer player you will need a body and make a post request to the /favorite/players route.

To update a favorite soccer player you will need a body and make a put request to the /favorite/players/:id route.

To delet a favorite soccer player you will need to make a delete request to the /favorite/players/:id route.
